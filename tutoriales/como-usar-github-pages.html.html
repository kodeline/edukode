Como publicar tu proyecto en GitHub Pages

Lo primero que debemos hacer es seleccionar el proyecto que deseamos subir, les recuerdo que el archivo index.html debe estar en la carpeta principal, luego debemos ir a la carpeta del proyecto a traves de la terminal Git Bash, una vez dentro de la carpeta del proyecto inicializamos el repositorio con el siguiente comando git-init ya inicializado SEGUIR VIENDO EL VIDEO desde el 1:16 https://www.youtube.com/watch?v=I-d9_l7myXw
